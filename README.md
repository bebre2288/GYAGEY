# GYAGEY

## English
Graphics Yet Another Game Engine. Yep.

## Quick Start

```console
$ make
$ ./gyagey
```

Game json file can be provided in first argument. Default is `game.json`

## Gameplay 

 - Start.
 - Settings.
 - press <r> to restart when game ended.
 - <Esc> to settings.
