# GYAGEY

## English
Graphics Yet Another Game Engine. Yep.

## Quick Start

```console
$ make
$ ./gyagey ./game.json
```
## Gameplay 

 - Start.
 - Settings (only sound mute for now).
 - press <r> to restart when game ended.
 - <Esc> to settings.
